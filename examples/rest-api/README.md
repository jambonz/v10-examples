# REST API Examples

Examples that use the jambonz REST API directly. These are runnable scripts (not servers) that demonstrate how to use the API for tasks like creating outbound calls, listing active calls, and managing conferences.

All REST API examples require `JAMBONZ_REST_API_BASE_URL`, `JAMBONZ_ACCOUNT_SID`, and `JAMBONZ_API_KEY` environment variables.
